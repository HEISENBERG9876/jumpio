# jumpio
A 2D platform game maker with procedural level generation

# How to run:
1. Download the builds/linux or builds/windows folder depending on the platform (or just clone the whole repository)
2. Run the executable in the correct folder, depending on the platform- builds/linux/Jumpio.x86_64 or builds/windows/Jumpio.exe

NOTE: First log in/register in the app will be very slow, since the server spins down after inactivity.

# Controls
LevelCreator:
WASD- camera movement
X- change mode from place to delete. X again to change back.
Left click- selecting object from toolbar, placing object, deleting object (after pressing X to change mode)
Z- Undo
C- Redo

Overall:
ESC- menu
WASD/Arrows/Space- player movement
